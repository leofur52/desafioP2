CRUD de Livros

Desafio - CRUD de Livros

Este projeto é parte do desafio proposto para substituir a prova do bimestre.

Objetivo

Desenvolver um sistema CRUD para gerenciamento de livros em um ambiente web.

Funcionalidades

  • Inserção de novos livros
  • Consulta de livros cadastrados
  • Atualização dos dados de um livro
  • Remoção de livros do sistema

Informações Técnicas

O backend do sistema será desenvolvido em JSP (Java Server Pages) e o banco de dados utilizado será MySQL Workbench.
